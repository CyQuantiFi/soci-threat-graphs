POR-004 · Refrigerated cargo monitoring tampering causing perishable losses

An attacker compromises the reefer-monitoring system that aggregates temperature telemetry from refrigerated containers awaiting shipment or in storage. By suppressing alarms or silently altering setpoints, the attacker causes cold-chain failures across a large reefer population. Loss event is high-value perishable cargo and class-action exposure. Pattern reflects general IoT-aggregator security weakness documented in maritime cyber reports.

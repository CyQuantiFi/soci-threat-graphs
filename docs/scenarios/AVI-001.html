AVI-001 · Baggage handling and ground systems disruption via shared IT services

An attacker compromises an airport's shared IT services environment, which hosts identity, file shares, and middleware used by airline ground handlers and baggage-handling-system (BHS) operators. The result is multi-airline ground-stop conditions even though no aviation-specific safety system is directly compromised. Pattern is consistent with disruptions seen at major international airports from common ground-systems outages and ransomware events affecting shared airport IT.

GAS-005 · SCADA protocol abuse (Modbus, DNP3) against gas distribution

An attacker who has established a foothold on the gas distribution control network uses native ICS protocols (Modbus TCP, DNP3) to issue commands to remote terminal units (RTUs) controlling pressure-regulating stations. Protocols lack strong authentication; commands appear legitimate. Consequence is unplanned pressure changes affecting distribution customers, particularly large industrial consumers and gas-fired generators.
